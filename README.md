# Image Text-to-Speech Converter

## Overview

This project extracts text from images using Optical Character Recognition (OCR) and converts the detected text into speech using Text-to-Speech (TTS) technology.

The solution enables users to transform printed or digital text from images into audio, improving accessibility and providing hands-free content consumption.

## Features

* Image-to-Text Extraction (OCR)
* Text-to-Speech Conversion
* Support for Multi-line Text Recognition
* Automated Processing Pipeline
* Audio Output Generation
* User-Friendly Workflow

## Project Workflow

1. Upload an image containing text.
2. Extract text using OCR techniques.
3. Process and clean the extracted text.
4. Convert text into speech.
5. Generate playable audio output.
